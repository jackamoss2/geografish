About GeograFish

GeograFish is a lightweight web app built to view basic GIS data. Users can create maps, add data, and customize visual properties. Maps can be shared via unique URL. GeograFish is not intended to be used to host or store data. Data cannot be manipulated by the app. Displayed data may not be accurate. The app should not be used for design. This app is built by Jack Amoss as a capstone project for PDX Code Guild's Full Stack bootcamp.
