How to use GeograFish to make a map

Note: At this time, you can only upload small .geojson (or .json) files. Other data formats are not supported.

  1. Create an account
  2. Create a map
  3. Upload some data
    4. Some sources: Cartograhy Vectors, PortlandMaps Open Data
  4. Customize the data layers visual appearance
  5. Share the map by copy and pasting the URL, or view your created maps on your user profile
